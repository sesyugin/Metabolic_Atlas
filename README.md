# Интерактивный атлас метаболических путей

> Образовательное веб-приложение для студентов медицинского вуза (биохимия, 2–4 курс).
> Визуализирует основные метаболические пути: интерактивная SVG-карта, пошаговый обзор с клиническими карточками, практические задания и режим преподавателя для аудиторий с большим экраном.

**→ [Открыть приложение](https://sesyugin.github.io/metabolic-atlas/)**

---

## Возможности

| Режим | Описание |
|-------|----------|
| **🗺 Интерактивная карта** | SVG-карта всех путей с pan/zoom, цветовым кодированием путей, кликабельными узлами и стрелками |
| **📚 Обзор** | Пошаговый просмотр пути с автовоспроизведением; карточки с ферментом, кофакторами, энергетикой и **клиническим значением** (ферментопатии, нозологии) |
| **🩺 Практика** | 3 типа заданий: восстановить путь, найти нарушенные реакции при ферментопатии, определить точки пересечения путей |
| **🖥 Преподаватель** | Защищён паролем; крупный UI для экранов 70"+; готовые сценарии уроков с вопросами для обсуждения и визуальным прогрессом слайдов |
| **🌙 Темы** | Светлая / тёмная тема, сохраняется в `localStorage` |
| **👋 Приветствие** | Обучающий попап при первом открытии сайта |

## Метаболические пути

- **Гликолиз** — 10 шагов, 3 необратимые регуляторные реакции
- **Пируват → Ацетил-КоА** — ПДГ-комплекс
- **Цикл Кребса** — 8 шагов, 3 CO₂, источник НАДН
- **β-Окисление жирных кислот** — активация, транспорт, цикл укорочения
- **Глюконеогенез** — обходные реакции ПК, ФФК-1, Г6Фазы
- **Обмен гликогена** — синтаза и фосфорилаза

## Стек

- **React 18** + **Vite 5** + **TypeScript** (строгая типизация, без `any`)
- **CSS Modules** (без внешних UI-библиотек)
- Все данные — локальные TypeScript-модули (без бэкенда, чистый SPA)
- Деплой: **GitHub Pages** через GitHub Actions

## Деплой на GitHub Pages

1. **Создайте репозиторий** на GitHub (например, `metabolic-atlas`).
2. В `vite.config.ts` убедитесь, что `base` совпадает с именем репозитория:
   ```ts
   base: '/metabolic-atlas/',
   ```
3. Включите GitHub Pages:
   **Settings → Pages → Source → GitHub Actions**
4. Сделайте `git push` в ветку `main` — Actions запустит сборку автоматически.

```bash
git remote add origin https://github.com/<username>/metabolic-atlas.git
git push -u origin main
```

Приложение будет доступно по адресу `https://<username>.github.io/metabolic-atlas/`.

```typescript
const HAS_VIDEO = true;
const VIDEO_EMBED_URL = 'https://www.youtube.com/embed/ВАШ_ID';
```

## Режим преподавателя

Доступ — по паролю **`1717`**. Режим оптимизирован для проекторов и экранов 70"+:
- Крупная типографика и кнопки навигации
- Визуальный прогресс-бар и счётчик слайдов (`1/5`)
- Карточки с вопросами для обсуждения
- Подсветка соответствующих метаболитов/реакций на карте
